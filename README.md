
# SmartNjangi

> Plateforme intelligente de gestion de tontines digitales avec scoring de credit alternatif et visualisation en realite augmentee

## Contexte

Projet de fin de cycle ingenieur - specialite Business Intelligence et Big Data.

Plateforme dediee au marche bancaire camerounais qui digitalise les tontines (njangis) et fournit aux banques partenaires un systeme de scoring de credit alternatif pour les populations non bancarisees.

## Stack technique

| Couche | Technologies |

|--------|-------------|

| Mobile | React Native (Android) |

| Web | React |

| Backend API | Node.js / Express |

| Base operationnelle | MongoDB |

| Big Data | Hadoop + Spark |

| Machine Learning | Spark MLlib |

| Realite Augmentee | AR.js + Three.js |

| Authentification | JWT |

| Mobile Money | MTN MoMo API |

| Notifications | Firebase Cloud Messaging |

## Structure du projet

- `docs/`       : Documentation, cahier de projet, diagrammes

- `backend/`    : API REST Node.js

- `mobile/`     : Application mobile React Native

- `web/`        : Dashboard web banque (React)

- `bigdata/`    : Scripts Spark et notebooks ML

- `ar-module/`  : Module AR.js / Three.js

- `data/`       : Donnees simulees

- `secrets/`    : Secrets locaux (NON commite sur Git)
